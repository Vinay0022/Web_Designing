Q13) What are Media Queries? Explain with Example.

Media Queries are used to make the website responsive across all the devices of different sizes.
    The Following code is for making a div responsive.
    < html >
    < head>
    < style>
div{
    margin: 50px;
padding: 30px;
font-size: 80px;
}
@media screen (max-length: 767px){
    div{
        padding: 10px
        font-size: 30px;
        margin: 20px;
    }
}
    < /style>
    < /head>
    < body>
< div>This is a Div Tag which is responsive.< /div>
    < /body>
    < /html>
